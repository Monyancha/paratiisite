# paratiisite

## Prerequisites

* npm/yarn
* gulp

## Installing

    $ git clone https://github.com/Paratii-Video/paratiisite
    $ cd paratiisite
    $ yarn/npm install

## Development

    $ gulp

## Deploy

    $ gulp deploy
